Python 条件语句是通过一条或多条语句的执行结果（`True` 或者 `False`）来决定执行的代码块，在python中，没有`switch`语法。

# 1.if语法格式

```python
mode = True

if mode :
    pass
else:
    pass

```
`注意：pass是一个空语句，一般用于占位符`


# 2.if嵌套

```python
if condition:
    pass
    if condition1:
        pass
    elif condition2:
        pass
    else:
        pass
else:
    if condition3:
        pass
    else:
        pass
```
