我们之前开发的都是业务代码，如果我们需要开发一些库代码给小伙伴们用或者给自己的团队用。那么如何使用webpack打包呢？这小节将带大家写webpack对库进行打包。

# 1.新建项目

由于library是要上传到npm或者其他包管理器上的，所以我们需要新建一个项目

```txt
webpackLibrary
```

#### 进入项目

```
cd webpackLibrary
```

#### 初始化项目

```
yarn --init
```
