# 2.标识符

标识符就是用于给java程序中变量、类、方法等名称符号

使用标识符时，需要遵守几条规则：
- 标识符可以由字母、数字、下划线（_）、美元符（$）组成，但不能包含 @、%、空格等其它特殊字符，不能以数字开头
- 标识符不能是 Java 关键字和保留字
- 标识符是严格区分大小写的
- 标识符的命名最好能反映出其作用，做到见名知意
