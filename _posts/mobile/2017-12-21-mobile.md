# 1.利用H5Api在移动端上传图片

```javascript
onUpload:function(e){
	let formData = new FormData();
	formData.append('file', e.target.files[0]);
	formData.append('type','test');  //添加测试数据
	axios.put(http.upload.img.url,formData).then(data =>{
		self.$data.userPhoto1 = data.data;
	}).catch(err =>console.log(err));
}


```
